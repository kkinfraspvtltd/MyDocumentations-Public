# Grafana Installation

## Grafana Cloud Setup

The steps below will show you how to set up Grafana Cloud. The key selling point for Grafana Cloud is that you don’t need to manage your own infrastructure to run Grafana. It is taken care of for you.

### 1. Sign up for Grafana Cloud

Visit the Grafana Cloud website:

https://grafana.com/products/cloud/

### 2. Review the Subscription Plans

Review the available subscription plans and select the one that aligns with your goals.

> **Note:** The **Free Tier** is recommended for basic exploration.

### 3. Explore the Demo Resources and Data

1. Using the left navigation bar, select **Apps → Demo Data Dashboards**.
2. Select a suitable demo.
3. Click **Install Dashboards**.

### 4. Explore the Dashboards and Panels

Based on what we discussed in Chapter One, explore the dashboards and panels.

For more information, see the **Get Started with Grafana Cloud** documentation:

https://grafana.com/docs/grafana-cloud/get-started/
