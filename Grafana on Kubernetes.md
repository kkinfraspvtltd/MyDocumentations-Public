# Grafana Installation

## How to Run Grafana on Kubernetes

The steps below will show you how to install and run Grafana on Kubernetes environments. This is mostly relevant if you plan to deploy Grafana to production on either a cloud or a self-hosted environment.

### Kubernetes Installation

You will need to have a running Kubernetes installation. How you install it depends on your use case. Follow the relevant instructions on the Kubernetes website:

https://kubernetes.io/docs/setup

### Install and Run Grafana

For the most up-to-date instructions, visit the Grafana website:

https://grafana.com/docs/grafana/latest/setup-grafana/installation/kubernetes/

You will need to create a namespace and apply the sample Kubernetes YAML file.

> **Pro Tip:** For production environments, it’s best to install using Infrastructure as Code (IaC) instead.

https://www.hashicorp.com/en/resources/what-is-infrastructure-as-code
